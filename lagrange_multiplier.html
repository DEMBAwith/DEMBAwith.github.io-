Lagrange未定乗数法(Lagrange multiplier)

Theorem

2変数実数関数$z=f(x,y):\mathbb{R}^2 \Rightarrow \mathbb{R}$が曲線$C(x,y)=0$上で極大値/極小値をとるとき、 \begin{eqnarray} L(x,y,\lambda) \equiv f(x,y)-\lambda C(x,y) \end{eqnarray} は次を満たす。 \begin{eqnarray} \frac{\partial L}{\partial x} = 0, \quad \frac{\partial L}{\partial y} = 0 \end{eqnarray} ただし、極値$(a,b)$において$C_x(a,b)$と$C_y(a,b)$の少なくとも一方が$0$でないとする。

Proof

極値を$(a,b)$として、$C_y(a,b)\neq 0$とする。陰関数の存在定理により、$x=a$近傍で$y=y(x)$が一意に定まり次を満たす。 \begin{eqnarray} y'(a)=-\frac{C_x(a,b)}{C_y(a,b)} \end{eqnarray} 点$(a,b)$で$df=0$なので次が得られる。 \begin{eqnarray} df &=& f_x(x,y) dx + f_y(x,y) dy \\ \Rightarrow 0 &=& f_x(a,b) + f_y(a,b) \left.\frac{dy}{dx}\right|_{(a,b)} \\ \Rightarrow 0 &=& f_x(a,b) - f_y(a,b)\frac{C_x(a,b)}{C_y(a,b)} \\ \Rightarrow 0 &=& f_x(a,b) - \lambda C_x(a,b) = \left. \frac{\partial L}{\partial x}\right|_{(a,b)} \end{eqnarray} 同様に$\left. \frac{\partial L}{\partial y}\right|_{(a,b)}=0$も示せる。
